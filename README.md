### **Customer Call List Data Cleaning**

####  **Using Pandas**



###### **Overview**



This project focuses on cleaning and preprocessing a customer call list dataset using Python and Pandas.



The goal is to handle missing values, remove duplicates, standardize formats, and prepare the dataset for further analysis.



###### **Dataset**



The dataset contains customer call records including:



Customer Name



Phone Number



Address



Call Status



Other relevant details



###### **Data Cleaning Steps Performed**



Removed duplicate records



Handled missing/null values



Standardized phone number formats



Cleaned address column (removed extra spaces)



Corrected inconsistent text formatting



Removed unnecessary columns

###### 

###### **Tools \& Technologies**



Python



Pandas



Jupyter Notebook



###### Purpose



To demonstrate practical data cleaning skills using Pandas, which is a key requirement for Data Analyst roles.
